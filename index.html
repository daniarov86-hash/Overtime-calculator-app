📖 מקרא פרמטרים
N
N — מספר שעות נוספות
סך כל השעות הנוספות שביצע העובד בתקופת החישוב. מוזן ידנית על ידי המשתמש.
X
X — ערך שעה נוספת (ערך ש.נ מהתלוש)
שווי כספי של שעה נוספת אחת. מופיע בתלוש השכר תחת הסעיף "ערך שעה נוספת".
a
a — מקדם גילום
מקדם המשתנה בהתאם לכמות השעות הנוספות: 1.0 / 1.1 / 1.3 / 1.75. ככל שיותר שעות — המקדם גבוה יותר.
S
S — גילום שעות נוספות
עד 50 שע׳: S = a · N · log₈(N)  |  מעל 50 שע׳: S = S₅₀ + 1.75·(N−50). כלומר 50 השעות הראשונות מחושבות עם הלוגריתם, וכל שעה מעבר ל-50 מחושבת לפי מקדם 1.75.
